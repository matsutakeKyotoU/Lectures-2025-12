# Lectures-2025-12 Sample Programs with ChatGPT

ここに掲載したJupyter Notebookプログラムは、講師がChatGPTと会話しながらVibe Codingで作成したものです。授業中のデモではリアルタイムでChatGPTにプログラムを生成させましたが、これらのプログラムは同様のシナリオで予め用意したサンプルプログラムとなっています。同じプロンプトでもChatGPTの出力はその時々で異なるため、デモ時のプログラムとは異なっていますし、皆さんが実行しても、また違う回答が返ってくると想定されます。

各プログラムの中、プロンプトで与えた指示・質問は**太字**で表示し、ChatGPTの回答はそれに続くテキストとしてなるべくそのまま再録してあります。

## 注意！
ChatGPTの答えは必ずしも正確ではありません。ここに掲載したプログラムはあくまでも参考として供するもので、その正確性を必ずしも保証するものではありません。講師はここに配置したプログラムの正確性をできる限りチェックしていますが、重要な不具合（バグ）が残されている可能性もあります。皆さんが手元にコピーして走らせたり改変したりするのは自由ですが、あくまでも自己責任でご利用ください。

## サンプル・プログラム

### BlackScholes.ipynb
- Black-Scholesのオプション価格式（配当なし）とリスク指標（Greeks）の計算を実装、グラフ表示
- オプション価格からボラティリティを逆算する（インプライド・ボラティリティを求める）方法を実装
- 米国株式市場からアップル社の個別株オプションの終値を取得し、ストライクごとのインプライド・ボラティリティをグラフ表示

### EfficientFrontier.ipynb 
- 日経225から時価総額の上位20銘柄を選び、Yahoo Financeから株価を取得、効率的フロンティアをグラフ表示
- 最小分散ポートフォリオ、並びにシャープ・レシオ最大（接点）ポートフォリオを求め、投資ウェイトをグラフ表示

### JGBYieldCurvePCA.ipynb
- 財務省の国債金利情報から国債利回りの履歴を取得、イールドカーブをグラフ表示
- PCA分析を行い、カーブの動きを主成分3つで表現しグラフ表示

### StockChart.ipynb
- Yahoo Financeから株価データを取得、チャートで表示

## Pythonについて
Pythonはデータ分析、AIを含め幅広い分野で使われているプログラミング言語です。書いたプログラムをその場ですぐに実行できる「スクリプト言語」の一種で、いろいろな機能を持つ別モジュール（ライブラリ）が豊富に存在し、それらを組み合わせて使うことで、いろいろなタスクを柔軟にこなすことができます。ライブラリとしては次が代表的なものです。
- データ分析：pandas, numpy, scipy
- AI：TensorFlow, PyTorch, SciKitLearn
- グラフ表示：matplotlib, plotly
- Web作成：Django, Flask

## Jupyter Notebookの実行環境について
自分のコンピュータにPython がインストールされていることが前提です。
- Pythonが環境に入っていなければインストール: https://www.python.org
	- バージョンは現在3.xx.xx (Stable Releaseで最もバンジョンの高いものを選べば良い）
	- データサイエンスのパッケージ Anaconda https://www.anaconda.com でもインストール可
	- インストール後、コマンドプロンプト/ターミナルを立ち上げ、`python`(または`python3`)と入力してpythonが起動すればOK
- Pythonと組み合わせて使うモジュール（ライブラリ）パッケージをインストール
	- Pythonをインストールすると、パッケージのインストーラ`pip`が付属している
	- コマンドプロンプト/ターミナルから`pip install パッケージ名`と入力　（パッケージ名は複数指定できる）
		- デモで使用したJupyter Notebookは`pip install notebook`でインストールできる
        - Pythonの実行時に「パッケージがない」エラーが出た場合には、その都度`pip install ...`して再実行すればよい
- Jupyter Notebookの実行
	- コマンドプロンプト/ターミナルから`jupyter notebook`または`python -m notebook`と入力
		- ブラウザ内でJupyter Notebookの画面が開く
